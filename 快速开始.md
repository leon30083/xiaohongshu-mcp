# 小红书MCP - 快速开始指南

## 🚀 一键启动

### 方法一：PowerShell脚本（推荐）
```powershell
# 在项目目录下运行
.\start-mcp.ps1
```

### 方法二：批处理脚本
```cmd
# 双击运行或命令行执行
start-mcp.bat
```

### 方法三：手动启动
```bash
# 开发模式
go run . server

# 或编译后运行
go build -o xiaohongshu-mcp.exe .
.\xiaohongshu-mcp.exe server
```

## 📊 检查服务状态

```powershell
# 使用脚本检查
.\start-mcp.ps1 -Check

# 或手动检查
curl http://localhost:18060/health
```

## 🔧 常用操作

```powershell
# 重启服务
.\start-mcp.ps1 -Restart

# 停止服务
.\start-mcp.ps1 -Stop

# 编译后启动
.\start-mcp.ps1 -Build

# 查看帮助
.\start-mcp.ps1 -Help
```

## 🎯 在Trae中使用

1. **确保MCP服务运行**：服务地址 `http://localhost:18060`
2. **自动连接**：Trae会自动使用项目配置 `.trae\mcp.json`
3. **可用工具**：9个小红书相关的MCP工具

### 可用的MCP工具：
- `mcp_xiaohongshu-mcp_check_login_status` - 检查登录状态
- `mcp_xiaohongshu-mcp_get_login_qrcode` - 获取登录二维码
- `mcp_xiaohongshu-mcp_list_feeds` - 获取用户发布内容
- `mcp_xiaohongshu-mcp_search_feeds` - 搜索小红书内容
- `mcp_xiaohongshu-mcp_get_feed_detail` - 获取笔记详情
- `mcp_xiaohongshu-mcp_user_profile` - 获取用户主页
- `mcp_xiaohongshu-mcp_publish_content` - 发布图文内容
- `mcp_xiaohongshu-mcp_publish_with_video` - 发布视频内容
- `mcp_xiaohongshu-mcp_post_comment_to_feed` - 发表评论

## 🔍 故障排除

### 服务无法启动
```powershell
# 检查Go环境
go version

# 检查端口占用
netstat -ano | findstr :18060

# 强制停止并重启
.\start-mcp.ps1 -Stop
.\start-mcp.ps1 -Restart
```

### Trae无法连接MCP
1. 确认服务运行：`.\start-mcp.ps1 -Check`
2. 检查配置文件：`.trae\mcp.json`
3. 重启Trae IDE
